# java-kbc-quiz-game
The KBC Quiz Game is a standalone desktop application built using Java and JavaFX, inspired by the popular TV show Kaun Banega Crorepati (KBC). The game provides an interactive quiz experience with multiple-choice questions, lifelines, scoring system, and a user-friendly GUI. 

Features:
Multiple-choice quiz with increasing difficulty
Lifelines (like 50-50, Audience Poll, etc.)
Score calculation and result screen
JavaFX GUI interface
Smooth navigation between screens
Works on any system with Java installed
